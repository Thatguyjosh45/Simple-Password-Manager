# Simple Password Manager 🔐

A lightweight, offline password manager with AES-256 encryption.

## Features
✅ AES-256 Encryption 🔐
✅ Secure SQLite Storage 💾
✅ CLI Interface 🖥
✅ GUI with Tkinter 🖱
✅ Password Generator 🔑
✅ Secure Master Password with Argon2 🔒
